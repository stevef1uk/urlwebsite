Welcome to SJFisher.com

I will use this site to learn about web analytics and to document my URL redirect SaaS service

My time limited URL redirect service is available at https://url.sjfisher.com

To use this service you first need to request an API key which can be done as follows:

curl -d '{"email": "your-valid-email-address" }' https://url.sjfisher.com/getkey

This returns: Record created {"key": " key-value " } {"Notice": "Please check for email and activate before using the key" }

Then you will receive an email that you need to click on the link in order to activate the returned API key

To create a short URL you need to do the following:

curl -d '{"Shorturl": "your-short-id", "From": "YYYY-MM-DDTHH:MM:SS+00:00", "Longurl": "the-target-url", "To": "YYYY-MM-DDTHH:MM:SS+00:00", "Defaulturl": "default-url"}' -H "Content-Type: application/json" -H "X-Api-Key: your-api-key" -v -X POST https://url.sjfisher.com

example:

curl -d '{"Shorturl": "steve", "From": "2020-12-31T00:00:00+00:00", "Longurl": "http://topgear.com", "To": "2021-12-31T00:00:00+00:00", "Defaulturl": "http://google.com"}' -H "Content-Type: application/json" -H "X-Api-Key: blkkuyjewpxr" -v -X POST https://url.sjfisher.com

This is will return

{"shorturl": " d41d8cd98f00b204e9800998ecf8427e " }

Please note that in the free tier (the defaut) the Shorturl has been converted into an Md5 hash and that value is returned

To access the short url use:

https://url.sjfisher.com?steve if on an elevated tier (email me for this) or https://url.sjfisher.com?d41d8cd98f00b204e9800998ecf8427e if not

Note: the d41d8cd98f00b204e9800998ecf8427e is the Md5 hash of steve and you will need to use the Md5 value returned for you

Outside of the From and To date and time periods the Defaulturl URL will be the redirect target. The Shorturl URL will only active within the To and From period

I have set a limit per email address of 100 record or updates

As there is a uniqueness test on the short url (even if it is turned into the Md5 hash) you may get an error response if it isn't unique.

There is also a profanity filter on the short url

Keys have a life of about a month but when expired you can request another as shown above

For the free tier the live duration of a record you have addeded is also one month, this can be changed if you ask for elevated rights

DISCLAIMER: As a free service there are no warranties produced and the service can be discontinued at any time

Personal data protection policy - I won't use your personal data (your email and URLs) for any reason other than to provide this service. The data is held encrypted in a Cloud SaaS database.

You can immediately delete your URL records by updating them with a From and To date in the past or leave them for a month after the To date to automtically be deleted

You can update your URL records in the same way as you created them, and apart from the Shorturl field any of the others can be changed

Your API Key will be automatically deleted 30 days after it was created

As an enhancement I will add a data cleanup endpoint in the future
