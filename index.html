Welcome to SJFisher.com

I will use this site to learn about web analytics and to document my currently URL redirect SaaS service

My time limited URL redirect service is available at https://url.sjfisher.com

To use this service you first need to request an API key which can be done as follows:

curl -d '{"email": "your-valid-email-address" }' https://url.sjfisher.com/getkey

This returns: Record created {"key": " key-value " } {"Notice": "Please check for email and activate before using the key" }

Then your will receive an email that you need to click on the link in order to activate the returned API key

To create a short URL you need to do the following:

curl -d '{"Shorturl": "your-short-id", "From": "YYYY-MM-DDTHH:MM:SS+00:00", "Longurl": "the-target-url", "To": "YYYY-MM-DDTHH:MM:SS+00:00", "Defaulturl": "default-url"}' -H "Content-Type: application/json" -H "X-Api-Key: your-api-key" -v -X POST https://url.sjfisher.com

example:

curl -d '{"Shorturl": "steve", "From": "2020-12-31T00:00:00+00:00", "Longurl": "http://topgear.com", "To": "2021-12-31T00:00:00+00:00", "Defaulturl": "http://google.com"}' -H "Content-Type: application/json" -H "X-Api-Key: blkkuyjewpxr" -v -X POST https://url.sjfisher.com

To access the short url use:

https://url.sjfisher.com?steve

I have set a limit per email address of 100 record or updates
